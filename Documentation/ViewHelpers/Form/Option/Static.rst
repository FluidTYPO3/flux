=========================================================
form.option.static ViewHelper `<flux:form.option.static>`
=========================================================


Form static caching option ViewHelper

Use this only when your Flux form is 100% static and
will work when cached.


.. _fluidtypo3-flux-form-option-static_arguments:

Arguments
=========


.. _form.option.static_value:

value
-----

:aspect:`DataType`
   boolean

:aspect:`Default`
   true

:aspect:`Required`
   false
:aspect:`Description`
   Configures caching of the DS resulting from the form. Default when used is TRUE which enables caching
